CSS - Cascading Style Sheets

É uma ferramenta de estilização de sites baseado em camadas de estilização - estilizar elementos individuais (como classes ou ids), agrupamentos de elementos (com classes ou estilização direta de elementos)

Existem diferentes formas de utilizar CSS

Estilização in-line

Quando o código do css está escrito no próprio elemento chamamos isso de CSS "inline" porque está "na linha" de código HTML que está estilizando, por exemplo, esse parágrafo.

Estilização "interna"

Estilização interna é a utilização da tag "styles" no próprio documento HTML para definir propriedades de CSS, nesse caso, a div "principal" é estilizada de forma "interna"

Estilização Externa

Muitas vezes convém utilizar estilização externa, para isso iremos utilizar da tag "linK" no head do documento fazendo referência ao arquivo de estilização que iremos utilizar, no caso aqui a referência é feita ao arquvi "aula4.css", arquivos de estilização possuem por padrão a extensão ".css"
