Exercício 1: Página de Perfil Pessoal
Objetivo: Criar uma página de perfil pessoal com navegação entre páginas e uma página adicional com uma tabela de habilidades.

Descrição:

Você já iniciou a construção de uma página de perfil pessoal. Agora, você deve adicionar elementos de navegação e expandir o projeto com novas funcionalidades. Siga os passos abaixo para completar o exercício:

1 - Página Principal (index.html):

Crie uma estrutura básica de HTML.
Adicione um <header> com um título (<h1>) para a página.
Inclua um <nav> no <header> com uma lista de links (<ul>, <li>) para navegação entre as páginas do projeto:
Página Principal (index.html)
Formulário (formulario.html)
Habilidades (habilidades.html)
Adicione um <section> contendo um <article> com um título (<h2>) e uma breve descrição sobre você.
Inclua um <footer> com um aviso de direitos autorais.


2 - Página de Formulário (formulario.html):

Crie uma nova página de HTML com uma estrutura básica.
Adicione um <header> com um título (<h1>) para o formulário.
Inclua um <nav> no <header> com uma lista de links (<ul>, <li>) permitindo navegação:
Voltar à Página Principal (index.html)
Formulário (formulario.html)
Habilidades (habilidades.html)
Dentro de uma <section>, crie um formulário (<form>) com alguns campos (como nome, email e mensagem).
Adicione um <footer> com um aviso de direitos autorais.


3 - Página de Habilidades (habilidades.html):

Crie uma nova página de HTML com uma estrutura básica.
Adicione um <header> com um título (<h1>) para a página de habilidades.
Inclua um <nav> no <header> com uma lista de links (<ul>, <li>) para navegação:
Página Principal (index.html)
Formulário (formulario.html)
Habilidades (habilidades.html)
Dentro de uma <section>, crie uma tabela (<table>) com as seguintes colunas:
Habilidade: Nome da habilidade que você pretende desenvolver.
Objetivo: O que você espera alcançar com essa habilidade.
Adicione um <footer> com um aviso de direitos autorais.



### Exercício 2: Criação de uma Página de Blog
Objetivo: Criar uma página de blog que utilize elementos semânticos e de estruturação para criar um layout organizado e visualmente agradável. Utilize o arquivo de imagem "exemplo.png" como referência para o layout.

## Requisitos:

Cabeçalho com Links de Navegação:
Utilize as tags <header> e <nav>.
Crie um cabeçalho que contenha um menu de navegação com links para outras páginas do site. Por exemplo: Início, Sobre, Contato.


Imagem Relacionada ao Tema do Artigo:
Utilize a tag <img>.
Insira uma imagem no artigo com a tag <img>, que deve ser relevante ao tema do blog. Não se esqueça de adicionar um atributo alt com uma descrição da imagem.


Seção Principal com Título e Artigo:
Utilize as tags <main> e <article>.
Crie uma seção principal usando a tag <main>. Dentro dela, adicione um <article> contendo:
Um título para o artigo usando a tag <h1> (ou <h2>, se apropriado).
O conteúdo principal do artigo, que deve estar dentro de tags <p>.


Seção de Links Relacionados:
Utilize as tags <aside>, <ul>, <li>, e <a>.
Crie uma seção lateral (<aside>) com uma lista de links relacionados. Esses links precisam apontar para páginas reais, mesmo que não tenham relação como tema.


Rodapé com Informações de Copyright:
Utilize a tag <footer>.
Adicione um rodapé à página com um aviso de copyright. Exemplo: © 2024 Meu Site.


Observações Adicionais:
As tags mencionadas são necessárias para cada requisito, mas você pode utilizar outras tags conforme necessário para tornar a página bem estruturada e visualmente agradável.
Certifique-se de que o layout da sua página se assemelhe ao desenho mostrado no arquivo "exemplo.png". O design deve seguir o esquema de layout e a disposição dos elementos apresentados na imagem.


Desafio:
Faça com que o layout de sua página se estruture da mesma forma como a do desenho em "exemplo.png". Preste atenção à posição dos elementos, tamanhos, margens, e outros detalhes visuais para replicar o design da imagem.
