Tabelas e seus usos

Tabelas eram mais comuns como forma de organizar a disposição do conteudo numa página porém com surgimento de novas ferramentas não é mais tão utilizando quanto já foi, mas ainda uma ferramenta bastante útil de se aprender...

Uma tabela é composta por quatro tags diferentes:

  • table: Que é a tag que define onde está a tabela no documento HTML
  • tr: Que define uma linha da tabela
  • th: Que define um cabeçalho da tabela
  • td: Que define a célula de uma tabela
